# k_markdown_parser

A lightweight and customizable Flutter Markdown renderer with dark mode support and built-in styling.

## Features

- Headers (#, ##, ###, ####)
- Bold, Italic, Strikethrough, Inline Code

## Usage

```dart
CustomMarkdown(
  data: '''
    # Heading
    Some **bold** and _italic_ and ~~strike~~ and `code`
    ''',
)
```
